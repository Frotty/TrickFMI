trickBall.fmu

The bouncingBall implements the following equation:
  • der(x) = v;
  • der(v) = a = force/m;
  • force = env_force * |x - origin|
with start values x=[5.0,5.0], env_force=8.0, origin=[0.0,2.0] and
  • x: position of ball [m], used as state
  • v: velocity of ball [m/s], used as state
  • der(x): velocity of ball [m/s]
  • der(v): acceleration of ball [m/s2]
  • force: external force on ball [N], a parameter
  • origin: external force on ball [N], a parameter
  • env_force: central force [N], a parameter
